# Trading Strategy Simulator

Aplikasi simulasi trading dengan integrasi TradingView webhook dan Binance API.

## Fitur
- Backend: 
  - Menerima sinyal TradingView via webhook
  - Validasi sinyal trading
  - Simulasi order dengan TP/SL
  - Log order simulasi

## Teknologi
- Express.js
- Firebase

## Instalasi Lokal
